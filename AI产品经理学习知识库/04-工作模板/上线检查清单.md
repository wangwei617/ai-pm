# AI 产品上线检查清单

## 1. 场景与价值

- [ ] 用户场景明确。
- [ ] 业务价值明确。
- [ ] MVP 范围明确。
- [ ] 有可量化成功指标。

## 2. 产品体验

- [ ] 输入引导清晰。
- [ ] 输出结果可编辑。
- [ ] 有错误和空状态。
- [ ] 有等待和进度反馈。
- [ ] 有用户反馈入口。
- [ ] 高风险操作有人工确认。

## 3. AI 能力

- [ ] Prompt 已版本化。
- [ ] 输出格式稳定。
- [ ] 模型选择有依据。
- [ ] 长文本处理方案明确。
- [ ] RAG 或知识库范围明确。
- [ ] Agent 工具权限明确。

## 4. 评测

- [ ] 有测试集。
- [ ] 有人工标准答案。
- [ ] 已评测准确率。
- [ ] 已评测幻觉率。
- [ ] 已评测引用准确率。
- [ ] 已记录错误案例。
- [ ] 达到上线质量门槛。

## 5. 数据与成本

- [ ] 埋点方案完成。
- [ ] 用户采纳率可统计。
- [ ] 单次任务成本可监控。
- [ ] 响应延迟可监控。
- [ ] 调用失败率可监控。

## 6. 安全与治理

- [ ] 敏感数据处理明确。
- [ ] 权限控制明确。
- [ ] 审计日志明确。
- [ ] 高风险输出有提示。
- [ ] 高风险动作有确认。
- [ ] 有事故处理流程。
