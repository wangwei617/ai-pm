# AI 信任触点清单模板

这份模板用于检查用户是否能信任、控制和纠正 AI 输出。

## 1. 功能名称

填写 AI 功能名称。

## 2. 信任触点

| 信任触点 | 要问的问题 | 当前方案 | 风险 | 改进动作 |
| --- | --- | --- | --- | --- |
| 源头透明 | 这段内容从哪里来，能否标引用 |  |  |  |
| 不确定性表达 | 低置信时是否坦白 |  |  |  |
| 可逆性 | 用户采纳后能否撤回 |  |  |  |
| 可控感 | 用户能否调整范围、风格、长度 |  |  |  |
| 失败尊严 | 拒答/失败时是否给下一步 |  |  |  |
| 可纠错 | 用户能否指出错误 |  |  |  |
| 可追溯 | 是否能追踪输入、输出、版本和来源 |  |  |  |

## 3. 引用与来源

- 是否必须展示来源？
- 来源展示到文档级、段落级还是字段级？
- 用户是否能展开原文？
- 来源是否有时间、版本、权限信息？

## 4. 拒答与澄清

| 场景 | 用拒答还是澄清 | 文案 | 下一步动作 |
| --- | --- | --- | --- |
| 检索为空 |  |  |  |
| 召回低置信 |  |  |  |
| 用户问题模糊 |  |  |  |
| 多个候选相近 |  |  |  |
| 权限不足 |  |  |  |

## 5. 用户反馈

反馈入口包括：

- 采纳。
- 重新生成。
- 编辑。
- 不准确。
- 缺少证据。
- 重复。
- 越权风险。
- 转人工。

## 6. 验收指标

- 输出采纳率。
- 编辑率。
- 拒答正确率。
- 澄清成功率。
- 引用点击率。
- 人工二审通过率。
- 纠错反馈率。
